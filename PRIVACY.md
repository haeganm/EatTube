# Privacy Policy

EatTube does not collect, sell, share, or transmit personal data.

## Data Use

EatTube runs locally in the browser on YouTube pages. It reads visible page content such as video links, titles, durations, thumbnails, view counts, and watched/progress indicators so it can choose a 20-60 minute recommendation.

## Local Storage

EatTube uses `sessionStorage` only for temporary per-tab state, such as whether the popup has already been shown. This data stays in the browser tab and is not sent anywhere.

## No Tracking

EatTube has no backend, analytics, tracking pixels, account system, advertising SDK, or remote code.

EatTube is not affiliated with YouTube, Google, or Alphabet.
